# Spatial Reasoning Benchmark

This project evaluates LLM-based spatial reasoning across three scaffold conditions:

- base/  → direct LLM reasoning
- tool/  → LLM + graph-analysis tools
- kg/    → LLM + tools + semantic knowledge graph

The benchmark includes local, global, counterfactual, and conceptual reasoning tasks over spatial connectivity graphs.



INSTALLATION
---------------------------------------------------------

Install dependencies:

pip install pandas matplotlib networkx requests openpyxl

Tested with Python 3.12.



CLAUDE API KEY
-------------------------------------------------------------

This benchmark uses the Anthropic Claude API.

Set your API key before running.

PowerShell:

$env:ANTHROPIC_API_KEY="your_api_key"

macOS/Linux:

export ANTHROPIC_API_KEY="your_api_key"



RUNNING THE BENCHMARK
--------------------------------------------------------------

Run Experiments + Analysis:

python main_benchmark.py

This will:
1. Run all benchmark experiments
2. Export CSV outputs inside the corresponding condition folders
3. Aggregate results
4. Generate final summaries and plots



Run Only Experiments
--------------------------------------------------

python main_benchmark.py --run-only

Runs all LLM experiments without performing final aggregation.



Run Only Analysis
--------------------------------------------------

python main_benchmark.py --analysis-only

Analyzes already-generated CSV outputs without making new API calls.

main_benchmark.py



AGGREGATION STRATEGY
-----------------------------------------------------------------------------

The analyzer uses only summary_by_question files as the source of truth.

Local tasks:
- Average question-level accuracy

Global tasks:
- Global Score =
  0.5 × metric accuracy
  + 0.5 × ranking accuracy

Counterfactual tasks:
- Ranking accuracy only

Conceptual tasks:
- Paraphrases are treated as repeated trials of the same question

PASS@K, PASS^K, and consistency are computed at the grouped question level.
